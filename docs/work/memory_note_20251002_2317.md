## Technical Discoveries

### Metal Integration Layers
```
whisper.cpp (C++)
    ↓
ggml backend API (C++)
    ↓
ggml-metal (C++ wrapper)
    ↓
ggml-metal-device.m (Objective-C)
    ↓
Metal.framework (native)
    ↓
ggml-metal.metal (MSL kernels)
```

Shaders can be **embedded** via `GGML_METAL_EMBED_LIBRARY=ON` → single binary deployment.

### Build Knowledge
```bash
# Static libs location
build/src/libwhisper.a
build/ggml/src/libggml*.a
build/ggml/src/ggml-metal/libggml-metal.a

# System dependencies (always present on macOS)
Metal.framework
Accelerate.framework
Foundation.framework
CoreGraphics.framework
```

### CGO Pattern (Reference)
```go
/*
#cgo LDFLAGS: ${SRCDIR}/vendor/whisper/libwhisper.a
#cgo LDFLAGS: ${SRCDIR}/vendor/whisper/libggml-metal.a
#cgo darwin LDFLAGS: -framework Metal
#include <whisper.h>
*/
import "C"
```
